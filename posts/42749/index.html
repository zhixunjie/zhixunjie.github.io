主题安装

git clone https://github.com/theme-next/hexo-theme-next themes/next

Next项目说明

  • 默认使用的模板引擎使用swig
  • 测试的时候有一个坑，测试时localhost使用的是http，而正式发布使用的是https，会造成测试和正式的情况不一样的假象。
  • 使用Next的版本库时需要注意：
    • 使用自定义的主题配置文件
    • 使用git子项目管理Next版本库
